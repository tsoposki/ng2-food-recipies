
# ng2-food-recepies

## Install
* [Git](http://git-scm.com/)
* [NodeJS](https://nodejs.org/en/)
* Clone project [repository](https://github.com/tsoposki/ng2-food-recipies)
  * Run: `npm install`


## Run:
    npm start
